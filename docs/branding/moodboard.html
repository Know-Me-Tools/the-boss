§ 06 — Usage Rules
Enforcement

The following rules are non-negotiable. They preserve the visual grammar that makes The Boss mark coherent at every scale and in every context.

Do
Use ember container (#FF6A3D) in dark mode. This is the "lit terminal" — the icon must glow against the dark background. 5.1:1 contrast ✓ AA
Don't
Never use a dark container on a dark background without a visible border. v1 failure: 1.05:1 contrast — the icon was completely invisible against the background.
Do
Crown (white solid) = active agents, passive (navy/ember ring on dark fill) = waiting. This two-tier visual grammar communicates agent state. Preserve it.
Don't
Never render the full icon below 32px with spokes visible. Below that threshold, drop spokes and render hex + crown dots only (r=3.5). At 16px: hex fill only.
Do
Use dark mode ember #FF6A3D on dark surfaces. Use light mode ember #E04E28 on light surfaces. These are calibrated for their respective backgrounds.
Don't
Never introduce a second accent color. The single ember + structural navy is the entire identity. Any additional hue — teal, purple, green — dilutes the mark irreversibly.
