Visual Language — Active vs Passive Agents

Active Crown (top 3 spokes)

Solid white fill. The three agents directly connected to The Boss. Authority, clarity, action. Color: #FFFFFF. Radius: 4.5 (primary), 2.75 (context).

Passive Agents (lower 3 spokes)

Dark fill with contrasting ring. Dark mode: #4A8EC7 navy ring. Light mode: #E04E28 ember ring. Communicates standby/waiting state. Ring blends subtly into orange hex in light mode — intentional “punched through” effect.

Size Thresholds

At 32px and below: omit all spokes, render hex + crown dots only (r=3.5). At 16px (favicon): hex shape alone — all internal elements disappear below 1px at this scale.

§ 06 — Enforcement

Usage Rules

Do
Use ember container (#FF6A3D) in dark mode. This is the “lit terminal”. The icon must glow against the dark background. 5.1:1 contrast ✓
Don’t
Never use a dark container on a dark background without a border. v1 failure: 1.05:1 contrast — the icon was invisible.
Do
Crown nodes (white) = active agents. Passive nodes = navy ring (dark mode) / ember ring (light mode). This visual grammar is intentional and must be preserved.
Don’t
Never render the full icon below 32px with spokes visible. The spoke geometry collapses to noise at sub-32px. Use crown-dots-only at 32px, hex-only at 16px.
Do
Dark mode ember: #FF6A3D. Light mode ember: #E04E28. Use the contextually correct value — they are calibrated for their respective backgrounds.
Don’t
Never introduce a second accent color. The single ember + structural navy palette is the entire identity. Teal, purple, or green dilutes it permanently and irreversibly.
