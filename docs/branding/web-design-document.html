Features Grid

Built different

Everything Cherry Studio has. Plus the infrastructure layer.

ART
Artifact Authoring

Create, edit, and manage artifacts in conversation. Full lifecycle — draft, revise, version, export.

CTX
Context Management

Window optimization, memory injection, chunking. Surreal Memory MCP built in.

SKL
Skills Selection

Browse, activate, and chain skills mid-conversation from the travisjames.ai skills registry.

A2A
A2A · AG-UI · A2UI

Protocol-native agent communication. Connect to UAR, MCP, and external runtimes without friction.

RTG
Model Routing

LiteLLM to 140+ providers. BYOK. Candle-vLLM local inference with TurboQuant compression.

OSS
Open Source

Full source on GitHub. travisjames.ai OSS license. First public release under The Boss brand.

§ 06 — Theming

Dark / Light Mode

Dark (Default)

Deep charcoal. Ember glow. Authority · Depth · Focus. The lit terminal.

Light

Off-white. Charcoal authority. Concrete · Light · Structure.
